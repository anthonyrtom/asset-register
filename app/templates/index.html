Please select a file to upload to calculate depreciation,please note that the file can be the following: {csv, txt, xlsx, xls}. If you are not sure of the format download please use the download link above on the menu Please note the following about the downloaded file:

  1. The date dmy represent the date format, d for the day, m for the month and y for the year, the accepted variatioans are {dmy, dym, mdy, myd, ydm, ymd}
  2. The decimal symbol used in this app is a dot (.)
  3. DepreciationMethod is either straight (for straightline) or reducing for reducing balance
  4. YearendMonthAsNumber represent the year end of the entity represented by numbers eg Jan being 1 and Feb being 2 and so forth
  5. DateSeparator represent your system date separator which can be (/,-, " ") note the last one is a space
  6. YearendDay means the last day of the financial year eg 28 meaning 28 of the month in the YearendMonthAsNumber column
  7. DepreciationYears means the number of years the asset will be depreciated over
